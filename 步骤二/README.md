## 2.购买虚拟空间

##### 购买地址:

X宝去搜索虚拟空间

这里我不推荐,看着买差不多的就行.

建议选择购买香港空间,国内访问也不慢,也不用备案,省去很多时间,国内的服务器购买都是要备案,很麻烦

![](/assets/Snip20170216_27.png)

##### 空间后台:

购买完成后,客服会给你一个后台,并给你一个解析地址,并告诉你解析方式.

![](/assets/Snip20170216_28.png)![](/assets/Snip20170216_32.png)

一般后台会包含:空间,数据库

![](/assets/Snip20170216_11.png)

我们只用看空间的后台,数据库不用去管
