# 买域名租空间经验分享

#### 说明:

分享一下我购买域名上传网站走过的坑

顺便写一个简单易懂的快速建个人小站流程

##### 我的花费:

bid后缀域名阿里云 9元 2年.    某宝香港主机虚拟空间  36元 1年.一共45元.

个人小站我尝试过的方式以下几种:

##### 第一种:完全免费的

```
    使用github page

   能力需求:会用git 上传, 关于git使用,可以看一下陈其林之前转发的廖雪峰git教程.
```

##### 第二种:花费域名钱

```
   github page + 域名 购买费用

  能力需求:跟第一种一样, 但是可以从自己的域名转入到github页面上.
```

##### 第三种:花费域名钱和虚拟空间费用

```
   阿里域名 + 虚拟空间

 这个是我要写的教程. 
```
